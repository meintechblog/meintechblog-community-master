Create the plugin directory structure and three PHP files:

**1. `community-master.php`** (plugin entry point, per D-04):
- Plugin header: `Plugin Name: Community Master`, `Text Domain: community-master`, `Version: 1.0.0`, `Requires PHP: 8.2`, `Requires at least: 6.6`
- Guard: `defined('ABSPATH') || exit;`
- Define constants: `COMMUNITY_MASTER_VERSION` = `'1.0.0'`, `COMMUNITY_MASTER_FILE` = `__FILE__`, `COMMUNITY_MASTER_DIR` = `plugin_dir_path(__FILE__)`
- `require_once` for all three includes files: `class-community-master.php`, `class-cpt-project.php`, `class-meta-boxes.php`, `class-admin-columns.php`
- Call `Community_Master::instance()` to boot the plugin
- `register_activation_hook(__FILE__, ...)`: calls `CM_CPT_Project::register()`, `CM_CPT_Project::add_capabilities()`, `flush_rewrite_rules()`, `update_option('community_master_version', COMMUNITY_MASTER_VERSION)` (per FOUND-03)
- `register_deactivation_hook(__FILE__, ...)`: calls `flush_rewrite_rules()` (per FOUND-03)

**2. `includes/class-community-master.php`** (singleton orchestrator):
- Class `Community_Master` with private static `$instance` property
- Public static method `instance(): self` returning singleton
- Private constructor that wires hooks:
  - `add_action('init', [CM_CPT_Project::class, 'register'])` for CPT registration
  - `add_action('init', [CM_CPT_Project::class, 'register_meta_fields'])` for meta field registration
  - Hooks for meta boxes and admin columns will be added by Plan 02 classes that self-register

**3. `includes/class-cpt-project.php`** (CPT + meta + capabilities):
- Class `CM_CPT_Project` with all static methods

- Static method `register(): void`:
  - Calls `register_post_type('community_project', [...])` with these args:
    - `'labels'` array with full set: name="Community Projects", singular_name="Community Project", add_new="Add New", add_new_item="Add New Project", edit_item="Edit Project", new_item="New Project", view_item="View Project", search_items="Search Projects", not_found="No projects found", not_found_in_trash="No projects found in Trash", menu_name="Community Master" (per D-04)
    - `'public' => false` (no frontend archive in Phase 1)
    - `'show_ui' => true`
    - `'show_in_menu' => true`
    - `'show_in_rest' => true` (prep for Phase 3 REST API)
    - `'capability_type' => 'community_project'` (per D-02)
    - `'map_meta_cap' => true` (per D-02, CRITICAL per research pitfall 1)
    - `'supports' => ['title', 'thumbnail']` (per FOUND-01, FOUND-02, FIELD-01, FIELD-03; NO 'editor' per D-01)
    - `'menu_icon' => 'dashicons-groups'`
    - `'menu_position' => 25`
    - `'has_archive' => false`

- Static method `register_meta_fields(): void`:
  - `register_post_meta('community_project', '_community_master_description', [...])` with `'show_in_rest' => true`, `'single' => true`, `'type' => 'string'`, `'sanitize_callback' => 'sanitize_textarea_field'`, `'auth_callback' => fn() => current_user_can('edit_community_projects')` (per FIELD-02, SEC-01)
  - `register_post_meta('community_project', '_community_master_github_url', [...])` with `'sanitize_callback' => 'esc_url_raw'`, same auth_callback (per FIELD-04, SEC-01)
  - `register_post_meta('community_project', '_community_master_installer', [...])` with `'sanitize_callback' => 'sanitize_text_field'`, same auth_callback (per FIELD-05, SEC-01)

- Static method `add_capabilities(): void`:
  - Define array of ALL 11 primitive capabilities (per D-02, research pitfall 2):
    `edit_community_projects`, `edit_others_community_projects`, `publish_community_projects`, `read_private_community_projects`, `delete_community_projects`, `delete_private_community_projects`, `delete_published_community_projects`, `delete_others_community_projects`, `edit_private_community_projects`, `edit_published_community_projects`, `create_community_projects`
  - Loop over `['administrator', 'editor']` roles, get each with `get_role()`, add all caps with `$role->add_cap($cap)`

- Static method `remove_capabilities(): void`:
  - Same cap list, same roles, calls `$role->remove_cap($cap)` for each
  - Used by uninstall.php

- Static method `get_capabilities(): array`:
  - Returns the capability array (shared between add/remove/uninstall)

Create the `includes/` directory before writing files.
